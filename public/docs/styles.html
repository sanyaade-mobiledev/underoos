settings/_config.sass

Config


Configure settings for Underoos and Compass for css output. Disabling some of these features will result in smaller code, but less overall support. See the Compass reference for other available settings.


Property Value Usage
$legacy-support-for-ie true Toggles ie7 hacks (supported by Compass and Underoos polyfills)
$legacy-support-for-ie8 true Toggles ie8 hacks (requires Compass)
$support-for-original-webkit-gradients true Toggles support for the the original webkit style gradients (requires Compass)
$experimental-support-for-mozilla true Toggles support for the -moz prefix (requires Compass)
$experimental-support-for-opera true Toggles support for the -o prefix (requires Compass)
$experimental-support-for-microsoft true Toggles support for the -ms prefix (requires Compass)

Notes:


  • Pro Tip! When compiling for iOS and Android, disable ie and experimental support for other browsers

Button Groups & Toolbars


Use button groups to join multiple buttons together as one composite component. Build them with a series of <a> or <button> elements.


Building off of the button group concept, combine multiple .button-group elements under a .button-toolbar container to form a navigation system.


Utilizing Bootstrap's Button plugin, you can control a button group or toolbars states by utilizing the data-toggle attribute. Passing either buttons-checkbox or buttons-radio will allow buttons within a group to behave similar to their input counterparts.

components/_dropdowns.sass

Dropdowns


Apply the data-toggle="dropdown attribute to a target link or button, then structure a dropdown navigation under a ul.dropdown-menu. Within the menu item, there should be a listing of links and optional dividers.


Dropdown's utilize a visuallyhidden and visuallyshown technique rather than a display: none|block to allow screen readers to still discover the inner content.


Dropdown's require Bootstrap's Dropdown plugin.


When the target link is floated right, the dropdown will automatically align it's dropdown menu accordingly. Dropdowns are designed to work with tabs, pills, navbars, button-groups and split buttons.


For elements outside of a list, typically in stand alone buttons, utilize the button-group structure for creating dropdowns. Split button dropdowns allow an action on the left while the dropdown on the right can be used for contextual menu items.

Clearfixins


Clearfixins force a containers height to adjust when it contains floated children elements.


There are two mixins available for fixing floats


  • +clearfixer The micro clearfix utilizing pseudo elements to clear floats
  • +clearfix Uses overflow: hidden to clear floats
